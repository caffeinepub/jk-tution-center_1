# JK Tuition Center - Frontend

A modern web application for JK Tuition Center built with React, TypeScript, and the Internet Computer.

## Features

- **Public Landing Page**: Showcases the tuition center with information about courses, announcements, and contact details
- **Student Dashboard**: Authenticated students can view available courses and announcements
- **Admin Dashboard**: Administrators can create, edit, and delete courses and announcements
- **Internet Identity Authentication**: Secure authentication using Internet Identity
- **Role-Based Access Control**: Different access levels for students and administrators
